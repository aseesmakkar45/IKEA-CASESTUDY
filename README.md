# 🛋️ IKEA Entrepreneurship Exploration Case Study

**[👉 View the Interactive Case Study Live Here](https://vaccasestudy.vercel.app/)**

An interactive, presentation-style React application built to explore the entrepreneurial journey, business model disruption, and global scaling strategy of IKEA. 

This project transforms standard academic case studies into an interactive digital dashboard with navigable slides, data visualizations, and strategic frameworks.

---

## 📖 About the Case Study

This case study delves into how Ingvar Kamprad disrupted the furniture industry by identifying a "Blue Ocean" opportunity and executing a flawless Cost Leadership strategy. It covers:

1. **Opportunity Recognition**: How IKEA targeted non-consumers (young families) in post-war Sweden during a housing boom.
2. **Business Model Innovation**: The shift from traditional retail markups to a self-service, flat-pack logistics model that drastically reduced costs.
3. **Value Chain & Logistics**: The famous 6-step IKEA value chain and how flat-packing eliminated the cost of "shipping air."
4. **Disruptive Innovation**: Navigating crises, including the Swedish cartel boycotts, and utilizing organizational ambidexterity for digital turnaround.
5. **ESG & Sustainability**: Strategic moves towards renewable energy and circular economy (furniture buy-back programs) to counter the "Fast Furniture Paradox."

---

## 🛠️ Technology Stack

This interactive presentation was built for speed, responsiveness, and clean aesthetics:

- **Frontend Framework**: React 18
- **Build Tool**: Vite (for rapid HMR and optimized building)
- **Styling**: TailwindCSS (for atomic, utility-first styling and rapid prototyping)
- **Routing/State**: Vanilla React hooks managing presentation slides and interactive data dashboards.

## 🚀 Setup & Installation

To run this interactive presentation locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/aseesmakkar45/IKEA-CASESTUDY.git
   cd IKEA-CASESTUDY
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
